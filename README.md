## Installation and Usage `Next.js project`

1. Install dependencies:

   - `npm install` for client and server parts

2. Start the application:
   - `npm run dev` for running the app

3. To build project:
   - `npm run build`

4. To start the production build:
   - `npm run start`

- Next.js
